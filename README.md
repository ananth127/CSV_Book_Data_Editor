# CSV Book Data Editor

## Project info

**CSV Book Data Editor** is a web app that allows users to upload, view, edit, filter, and download CSV files containing book data.

---

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

---

## Installation

```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm install
npm run dev

```
